# Web Tech Starter

PHP 8.2 + Apache + MySQL with Docker Compose

## Steps

1. **Start containers**
   ```bash
   docker-compose up -d
   ```

2. **Access application**
   ```
   http://localhost:8000
   ```

3. **Stop containers**
   ```bash
   docker-compose down
   ```

## Database

- User: `myuser`
- Password: `mypassword`
- Database: `mydatabase`

## Details

- Add PHP files to `src/`
- MySQL runs on port `3306`
